The week in Montage

Last week featured a number of small bug fixes in Montage for the button, blueprints, flow and an update to Mr v0.13.3. The translate composer now supports the standard wheel event. There was also a large update to Montage's MediaController which paved the way for a brand new video player in Digit!

You can see Montage in action on the TodoMVC website, and last week Mike Czepiel updated the example to Montage v0.13.
